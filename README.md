## hs-docker-sample
Sample for running
<a href="https://www.haskell.org/" target="_blank">Haskell</a>
app on docker

### usage
```
$ docker build -it <container-name>:<tag> .
$ docker run -it <container-name>:<tag>
Hello tatatakky !!
```
